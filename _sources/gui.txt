GUI
===

How would you do a cross-platform GUI in D?

There is `DWT <https://github.com/d-widget-toolkit/dwt>`_,
a pure D framework,
which ports the Java SWT API.

Then there are bindings to other frameworks
like `GTK+ <http://gtkd.org/>`_ or
`Qt <https://bitbucket.org/qtd/repo>`_.

   `Getting Started with Gtkd <http://britseyeview.com/software/articles/gsgtkd101.html>`_
